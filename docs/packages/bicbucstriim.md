# BicBucStriim

[BicBucStriim](https://projekte.textmulch.de/bicbucstriim/) streams e-books from a Calibre library to your e-book reading apps.

## Default Login Credentials

- **Username**: `admin`
- **Password**: `admin`

## Library Permissions

Ensure the user `sc-bicbucstriim` has read/write access to the Calibre library directory on your DiskStation before setting up BicBucStriim.
