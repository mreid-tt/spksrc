## Notes for Creating Screenshots

1. **Consistency:** Use DSM 7.x screenshots where possible (most users are on DSM 7)
2. **Resolution:** Capture at reasonable resolution (1920x1080 or similar)
3. **Cropping:** Crop to show only relevant UI elements
4. **Highlighting:** Consider adding red boxes/arrows to highlight key areas (optional)
5. **Privacy:** Remove/blur any personal information (usernames, IP addresses, etc.)
6. **Format:** Save as PNG for UI screenshots
